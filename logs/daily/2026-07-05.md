## 关键认知

1. RAG 本质 = 构建高质量 Prompt；所有检索操作都是为了找到最相关的"参考资料"
2. Embedding 模型必须索引和查询统一，否则向量空间不匹配（最常见隐性 Bug）
3. 混合检索：向量（语义）+ BM25（精确关键词）+ RRF 融合 → 工业级必备
4. 我的项目是 **Advanced RAG**（混合检索 + Citation + 拒答），Forge 是 **Agentic RAG**
5. Agent 评估 7 维度：任务结果、工具动作、过程轨迹、依据一致性、安全、效率、用户体验
