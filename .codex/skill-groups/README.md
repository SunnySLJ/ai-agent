# Skill Groups

本目录是功能分组索引，不是运行时 skill 安装目录。

活跃 skill 仍保留在 `.codex/skills/<skill-name>/SKILL.md`，以保证 Codex/Claude 等运行时可以按平铺命名空间发现。已合并或不再推荐的新任务入口放在 `.codex/skill-archive/`。

| 分组 | 说明 |
|---|---|
| `00-core-flow` | 总调度、skill 规则、项目指导文件、复盘进化 |
| `10-research-prd-architecture` | 脑暴、调研、PRD、架构基线 |
| `20-design-frontend` | UI 设计、原型迁移、Next.js 前端施工 |
| `30-implementation` | TDD 实施与 Spec-Kit implement |
| `40-testing` | 后端、前端、接缝、完整链路测试闭环 |
| `50-spec-kit` | Spec-Kit 文档链、constitution、Git 辅助 |
| `60-handoff-tools` | AI 交接提示词、代码链路 handoff、API handoff、AI API 工具、视频剪辑 |
| `90-archive` | 已归档旧入口映射 |
